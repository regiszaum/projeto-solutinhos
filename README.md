# Projeto Solutinhos

## Rodar projeto local (Windows)

npm i @vue/cli-service

npm install

npm run serve
## Rodar projeto local (Em produção)
npm run build

## Rodar lint
npm run lint

### Editar configurações:
Veja em: [Configuration Reference](https://cli.vuejs.org/config/).
